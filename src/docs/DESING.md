# Design System



## Core Principles



This design system is built around a dark aesthetic, prioritizing clarity and a modern feel.



## Thematic Palette



Our palette is designed for high fidelity across all components, featuring:

* **Primary Color**: `#009bdb` - A vibrant blue for key interactions and branding.

* **Secondary Color**: `#577b95` - A supportive blue-gray for secondary elements.

* **Tertiary Color**: `#d17e0f` - An accent orange for highlights and emphasis.

* **Neutral Color**: `#73787c` - A balanced gray for backgrounds and text.



## Typography



Our typography system ensures readability and visual hierarchy:

* **Headlines**: Utilizes 'Manrope' for a clean, contemporary feel.

* **Body Text**: Set in 'Inter' for excellent legibility at all sizes.

* **Labels**: Also uses 'Inter' for consistency in interactive elements.



## Shape & Form



### Roundedness

Components feature maximum roundedness, giving elements a soft, pill-shaped appearance.



### Spacing

Elements are designed with normal spacing, providing a balanced layout without feeling too dense or overly spacious.
